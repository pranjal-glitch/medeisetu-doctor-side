# Medisetu Doctor Dashboard

A simple and interactive web-based dashboard designed for doctors to manage patient appointments, view patient profiles, and maintain patient history – built using pure HTML, CSS, and JavaScript.



## 🚀 Project Overview

Medisetu Doctor Dashboard is a front-end application that allows doctors to:

- Log in using a Doctor ID  
- View daily appointments  
- Book new appointments  
- Open patient profiles  
- Add and save patient medical information  
- View patient history  
- Logout securely  

All data is stored using **LocalStorage**, so no backend is required.


## ✨ Features

### 🔐 Authentication
- Simple Doctor ID login  
- Persistent session using LocalStorage  
- Logout functionality  

### 📅 Appointments Management
- View list of booked appointments  
- Add new appointments  
- Store appointment details locally  

### 👤 Patient Profile
- View patient details  
- Save medical notes  
- Record vitals (BP, Pulse, Temperature)  
- Maintain patient history  

### 💾 Data Persistence
- All information stored in browser LocalStorage  
- Data remains after page refresh

## ▶️ How to Run the Project

1. Download or clone the repository  
2. Open the project folder  
3. Double click on **index.html**  
4. The application will open in your browser  

No server or installation required.

---

## 📌 How to Use

### 1. Login
- Open the app  
- Enter any Doctor ID  
- Click **Continue**

### 2. Appointments
- View booked appointments  
- Click **Book Appointment** to add a new one  

### 3. Patient Profile
- Click **View Patient** from appointments  
- Add medical info and vitals  
- Save patient history  

### 4. Logout
- Click **Logout** to end session
